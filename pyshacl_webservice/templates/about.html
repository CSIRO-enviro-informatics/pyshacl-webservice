About

This web application is a form that lets you use pySHACL, a Python implementation of the Shapes Constraint Language (SHACL) standard to validate RDF data.

SHACL requires that validation constraints (cardinality, dependence, particular values etc.) are themselves expressed in the form of an RDF graph called a 'shapes graph'.

This web form expects a data graph to validate and a shapes graph to validate with (they may be the same file), both of which it then submits to pySHACL running as a Python module behind the scenes with validation results displayed to the user.

SHACL defines the format of a validation process, see the SHACL standard's Validation Report Section.
